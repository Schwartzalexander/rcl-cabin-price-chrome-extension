# Store Description

Cruise Cabin Price & Sort makes Royal Caribbean and Celebrity Cruises search results easier to compare by adding practical cabin-level pricing directly to the cruise cards.

The official search results usually show an average price per person. That is useful, but it can make it harder to compare the real cost of a trip when you are browsing different itineraries, durations, or guest counts. This extension calculates the estimated total cabin price and the cabin price per night from the visible price information on the page, then displays those values next to each loaded cruise result.

The extension also adds an extra sorting panel so you can sort the currently loaded cruise cards by the numbers that are often most useful when comparing cruises: cabin price per night, total cabin price, cruise duration, and original order. If enabled in the settings, it can also show and sort by price per person per night.

Key features:

- Supports Royal Caribbean and Celebrity Cruises search result pages.
- Adds an estimated total price per cabin to each cruise card.
- Adds an estimated cabin price per night.
- Optionally shows price per person per night.
- Adds extra sorting options for loaded cruise cards.
- Can sort by cabin price per night, total cabin price, cruise duration, or original order.
- Includes buttons to load more results and re-apply sorting after additional cruises are loaded.
- Automatically reads the guest count from the page when available.
- Allows a manual guest count when the page does not show occupancy clearly.
- Stores preferences locally using browser sync storage.

Important notes:

- The extension calculates values from prices already shown on the cruise line website.
- Promotions, taxes, fees, and included discounts are handled only insofar as they are already reflected in the visible price on the page.
- Final booking prices can still change depending on sail date, cabin category, availability, discounts, taxes, fees, and booking flow details.
- Sorting only applies to cruise cards that are currently loaded in the page. Use the extension's load controls or the website's own loading controls to load more results before sorting all visible cruises.

Privacy:

This extension does not collect, transmit, or sell personal data. It reads visible cruise result information from supported cruise search pages in order to calculate and display additional prices. Settings such as manual guest count and display preferences are saved with Chrome storage.
